# Work Planner

업무를 우선순위·마감순으로 계획하고, 반복 업무는 템플릿 버튼으로 빠르게 추가하는 모바일 반응형 웹앱.

## 주요 기능

- **업무 종류 관리** — 사용자가 직접 추가/수정/삭제
- **업무 CRUD** — 제목만 입력해도 등록 (우선순위 기본 3, 상태 기본 대기)
- **반복 업무 템플릿** — 버튼 클릭 한 번으로 오늘 업무 즉시 생성
- **붙여넣기 일괄 등록** — `제목 | 종류 | 우선순위 | 마감 | 메모` 양식으로 여러 업무를 한 번에 등록
- **출처 구분** — 템플릿으로 만든 업무는 `[반복]`, 직접 추가/일괄등록한 업무는 `[직접]` 라벨 표시
- **출처 필터** — 업무 목록에서 전체 / 반복만 / 직접만 필터링 가능
- **AI용 양식 복사** — AI에게 목록 만들어 달라고 할 때 쓰는 프롬프트 복사
- **AI에게 조언받기** — 현재 업무 목록을 텍스트로 복사해서 AI 조언 요청
- **계획 보기** — 우선순위 높은 순 → 마감 임박 순 정렬, 기한 초과 빨간 강조
- **지금 할 일** — 최상단에 1순위 업무 크게 표시
- **필터 기억** — 마지막 사용한 필터가 다음 방문에도 유지됨

## 로컬 실행

```bash
cd apps/work-planner
pip install -r requirements.txt
uvicorn src.main:app --reload --port 8000
```

- 앱: http://localhost:8000
- Swagger API 문서: http://localhost:8000/docs

## 테스트

```bash
cd apps/work-planner
pip install -r requirements.txt
pytest tests/ -v
```

## 업무 일괄 등록 양식

```
제목 | 종류 | 우선순위(1~5) | 마감(YYYY-MM-DD) | 메모
증례 포스팅 초안 | 증례 | 5 | 2026-06-18 | 활짝웃는
블로그 1건 | 블로그 | 3
```

뒤 항목은 생략 가능합니다. 생략 시 우선순위 3, 상태 대기로 등록됩니다.
